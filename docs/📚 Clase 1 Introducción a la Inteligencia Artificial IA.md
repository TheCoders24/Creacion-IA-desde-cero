## **📚 Clase 1: Introducción a la Inteligencia Artificial (IA)**

### **1. ¿Qué es la Inteligencia Artificial (IA)?**
La **Inteligencia Artificial (IA)** es un campo de la computación que permite a las máquinas simular capacidades humanas como el aprendizaje, el razonamiento y la toma de decisiones.

📌 **Tipos de IA**:
1. **IA débil (Narrow AI)**: Diseñada para tareas específicas (Ejemplo: Siri, Alexa).
2. **IA fuerte (Strong AI)**: Capaz de razonar y resolver problemas generales como un humano (Ejemplo: Modelos avanzados como GPT).
3. **Superinteligencia Artificial (ASI)**: Un nivel hipotético donde la IA supera la inteligencia humana.

### **2. Subcampos de la IA**
Dentro de la IA existen varios subcampos, pero los más importantes para crear modelos desde cero son:
- **Machine Learning (ML)**: Enseña a las máquinas a aprender patrones a partir de datos sin ser programadas explícitamente.
- **Deep Learning (DL)**: Usa redes neuronales profundas para procesar información y aprender de forma más avanzada.
- **Procesamiento del Lenguaje Natural (NLP)**: Permite que las computadoras entiendan y generen texto (Ejemplo: Chatbots, traductores).
- **Visión por Computadora**: Habilidad de interpretar imágenes y videos.

### **3. ¿Cómo funciona un modelo de IA?**
Un modelo de IA sigue un proceso general:
1. **Recolección de datos** 🏗️: Se obtiene información de calidad para entrenar el modelo.
2. **Preprocesamiento de datos** 🧹: Se limpian y organizan los datos para que el modelo pueda entenderlos.
3. **Elección del modelo** 🛠️: Se selecciona el algoritmo adecuado (red neuronal, regresión, etc.).
4. **Entrenamiento del modelo** 📊: Se ajustan los parámetros del modelo con los datos de entrenamiento.
5. **Evaluación del modelo** ✅: Se mide la precisión con datos que no ha visto antes.
6. **Optimización y ajuste** 🔧: Se mejora el modelo ajustando hiperparámetros.
7. **Despliegue** 🚀: Se implementa el modelo en una aplicación o sistema.
